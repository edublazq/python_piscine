# Python Piscine

Este repositorio contiene los ejercicios de la Piscine de Python de 42 Madrid.

## Cambio del Common Core

A partir de 2024, 42 Madrid ha modificado su programa educativo, reemplazando la tradicional Piscine de C con una Piscine de Python como punto de entrada al Common Core. Este cambio busca:

- **Facilitar el acceso**: Python es un lenguaje más accesible para principiantes, permitiendo a los estudiantes centrarse en conceptos de programación fundamentales sin la complejidad inicial de la gestión manual de memoria.
- **Modernizar el currículum**: Python es uno de los lenguajes más utilizados en la industria actual, especialmente en áreas como data science, machine learning, automatización y desarrollo web.
- **Preparar para el futuro**: Los estudiantes adquieren habilidades inmediatamente aplicables mientras desarrollan las bases necesarias para aprender otros lenguajes en el Common Core.

## Estructura del Proyecto

Este repositorio está organizado por días/módulos de la Piscine, cada uno enfocado en diferentes aspectos de la programación en Python.

## Requisitos

- Python 3.x
- Git

## Uso

Cada ejercicio contiene su propio directorio con instrucciones específicas. Sigue las guías de cada módulo para completar los ejercicios.
